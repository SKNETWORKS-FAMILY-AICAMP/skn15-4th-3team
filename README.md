## 프로젝트 확장성

* 다양한 보험 상품으로의 적용 가능
  * 건강보험, 화재보험 등 다양한 약관을 반영하면 챗봇의 목적을 바꿀 수 있게 됩니다.
* 판례 데이터 반영
  * 파인 튜닝(Fine-Tuning)을 통한 판례 기반의 추론 강화
  * 법적 기반을 통한 방향성 확보
  * 챗봇이 더욱 풍부한 역할 가능
